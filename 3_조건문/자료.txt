조건문(조건을 통과해야만 실행하는 구문)

if(조건 a){
    실행문a
}else if(조건b){
    실행문b
}else{
    실행문c
}

★ if는 필수로 필요 (1개만 사용가능)
  else if는 필요한 경우에만 사용 (복수로 사용가능)
  else도 필요하면 사용 (1개만 사용가능)

  비교연산자 (2개의 값을 비교하는 연산자)

> 크다
< 작다
>= 크거나 같다
<= 작거나 같다
== 같다 (equal)
!= 다르다/같지않다 (not equal)
